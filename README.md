# Dwoodline - Architectural Heritage Portfolio

A premium, responsive portfolio website for Dwoodline Architectural Heritage Experience built with Next.js-inspired design patterns and Tailwind CSS.

## 🎨 Features

- **Snap-Scroll Navigation**: Smooth section-by-section scrolling with animations
- **Premium Design**: Luxury brand aesthetic inspired by Rolls-Royce and Minotti
- **Responsive Layout**: Optimized for desktop and mobile devices
- **Interactive Portfolio**: Project showcase with image galleries
- **Smooth Animations**: Fade-in effects and staggered animations

## 📄 Pages

- `01-hero.html` - Hero section with video background
- `02-heritage.html` - Brand heritage and history
- `03-expertise.html` - Services and expertise
- `04-portfolio.html` - Project portfolio with image galleries
- `05-inquiry.html` - Contact and inquiry form

## 🚀 Deployment

This site is deployed on GitHub Pages. Any changes pushed to the `main` branch will automatically update the live site.

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/ralph_daccache/dwoodline.git
   cd dwoodline
   ```

2. Open any HTML file in your browser (all pages are static)

3. Navigate between pages using the menu

### Deploy Changes

1. Make your changes locally
2. Stage and commit:
   ```bash
   git add .
   git commit -m "Update: description of changes"
   ```
3. Push to GitHub:
   ```bash
   git push origin main
   ```

The site will update automatically on GitHub Pages!
